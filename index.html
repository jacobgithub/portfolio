Link: Infinity Work Clock

Code: Git/CodePen

Building Blocks: HTML/Bootstrap/CSS/JavaScript/jQuery

Description:

This was a project made for freeCodeCamp.com. It is a 'Tomato Timer' or 'Pomodoro Clock', but can realistically be altered into many different types of applications requiring a timer.

The Pomodoro Technique is a time management method developed by Francesco Cirillo in the late 1980s. The technique uses a timer to break down work into intervals traditionally 25 minutes in length, separated by short breaks....The method is based on the idea that frequent breaks can improve mental agility. - https://en.wikipedia.org/wiki/Pomodoro_Technique

Link: Random Quote Machine

Code: Git/CodePen

Building Blocks: HTML/Bootstrap/CSS/JavaScript/jQuery

Description:

This was a project made for freeCodeCamp.com. It is a 'Random Quote Generator'.

More accurately this project is a random item/element/object selector, I chose the specific styling based off of fandom of video games. This project could easily be made into selecting any sort of element to display random, such as pictures, videos, sounds. Another application idea I like is would be a combination of this & the timer application to create a 'Random Workout'.

Link: Animal Emergency Center

Code: Git

Building Blocks: HTML/CSS/jQuery/responsiveslides.js

Description:

This was a project made for Animal Emergency Center, an emergency veterinary hospital. It is a responsive redesign of their original site.

This is an example of how to migrate a site into a responsive design, the original site can still be found: here. This is also an example of creating a responsive design without a template/plugin such as bootstrap, all CSS is 'from scratch'. On the other end of the spectrum, it also showcases the use of jQuery plugins, with the responsiveslides.js plugin.

Link: Blog

Code: Git

Building Blocks: Python/Jinja/Google App. Engine/HTML/CSS/

Description:
This project was built using Google App. Engine for Udacity's CS 253 - Web Development.
This is a full blown, hand written web application written in Python and uses the Jinja2 template engine. The 'blog' section supports user registration, login/logout, and blog posts.
There are also links to some other fun applications using the same building blocks, such as an ASCII art board, which is essentially just a board where users can post text, and a simple ROT13 Cypher; which takes a string and 'encrypts' it with a simple rotation of thirteen alphabet characters.
